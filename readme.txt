FortressWP is a modern all-in-one WordPress security plugin combining 
firewall, malware scanning, AI-assisted analysis, and TOTP 2FA.

== Description ==

FortressWP provides:

* AI-assisted malware detection  
* File scanner with signature-based detection  
* Hourly/daily background scanning  
* TOTP 2-factor authentication  
* IP & user-agent firewall  
* Login brute-force protection  
* Automated signature updates  
* External blocklist integration  
* Audit logging  

== Installation ==

1. Upload the fortresswp.zip file to WordPress › Plugins  
2. Activate **FortressWP**  
3. Open the **FortressWP** admin menu  
4. Configure firewall, scanning, AI, and TOTP settings
